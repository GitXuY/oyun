Detailed Description

Class which encapsulates wxGraphicsContext, image, and SVG drawing.

This class accepts rectangle, pen, and text drawing commands, and draws its output to a wxMemoryDC used to make a bitmap image, a wxGraphicsContext used for on-screen display, and an SVG buffer used to create an SVG file.

float  y,
bool  right,
const wxString &  text 
)

Draw some text on the graph in the current font.

Text will be drawn with the current pen on the graph, but will be printed as black in the SVG file.

If right is set to true, the point (x, y) will be considered the center-right point of the text to draw. Otherwise, the point (x, y) will be considered the top-center point of the text to draw.

Parameters:
xX-coordinate of text, between 0 and 1
yY-coordinate of text, between 0 and 1
