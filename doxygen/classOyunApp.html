Detailed Description

The main Oyun application class.

In usual wxWidgets style, this class constitutes the program entry point. It creates the wxWizard and the various pages, links them together, and starts the wizard. When the wizard is closed, execution terminates.

void OyunApp::ShowHelp ( wxWindow *  parent = NULL,

Show a help page by filename.

This function opens the online help to a specific page. By default, filename is an empty string, which will show the help's contents. Note that the filename expected here is an internal .html file, specified as a relative path from the root of the documentation tree.

Parameters:
parentThe window to be the parent of the help dialog
filenameThe filename within the documentation tree to open. If empty, show the help contents.
