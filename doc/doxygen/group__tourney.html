Detailed Description

This module contains the match and tournament code. A Match is a controlled set of five games (or occasionally one) between two given players. The two tournament types (Tournament and EvoTournament) are collections of matches between a given set of players. The Tournament class is a round-robin tournament with one winner using five-game matches, while the EvoTournament is an "evolutionary" tournament in which the score of each player after each match determines each player's fraction in the next "generation" of players.

GenerationWeights

A map which associates player names (wxString values) with population fraction.

These population-fraction values are always between 0.0 and 1.0, and the contents of any GenerationWeights map always sum to 1.0 (100% of the population).
